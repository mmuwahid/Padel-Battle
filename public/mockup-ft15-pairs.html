Notes for Review

1. Pair avatars: stacked overlapping (-12px / -10px / -8px depending on size). Always 2 circles at .pr-pavi sizes.

2. Auto-fallback name: when pair has no name, shows "Player A / Player B" via existing formatTeam() helper. .pr-pcell-mb still shows individual names for clarity.

3. Color accent: optional per-pair color (preview only — defaults to accent green if unset). Set when admin creates pair.

4. ELO seed at 1500: Premier-Padel style — independent of individual ELOs. Stored on pairs.elo column, updated transactionally on match insert.

5. Lock-after-matches: format toggle disabled in edit sheet once season has matches. Pair delete button greyed out (.4 opacity) when pair has any matches.

6. YOU row: .pr-row.you tint applies when current user is one of the two pair members.

7. Season pill: shows "PAIRS" gold badge after season name when format='pairs'. All other seasons (individual) listed too — no filter (Q10).

8. MOTM still per-player (Q9) — the existing MOTM section in match cards continues to show 1 player from the 4 in the match.
