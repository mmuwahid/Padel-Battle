Notes for Review

1. Avatar treatment: organizer ring is gold, participants accent green, locked-state participants tinted by team (A green / B gold). Empty slots = dashed border + "+" symbol.

2. Status pill: dot+text mirrors live `.bdgn`/`.bpend` pattern. Open=accent dim, Locked=gold dim, Cancelled=danger dim.

3. Locked card actions: Reshuffle button only visible to organizer. "Log Score" deep-links to LogMatch with team_a/team_b pre-filled and disabled.

4. Auto-cancel: rendered with .cancelled modifier (opacity .55 + saturate .4). Note in footer explains why. After 24hrs the card hides from view; admin can purge from PlatformAdmin.

5. Notes field: optional, italic mono, only renders if organizer added one.
